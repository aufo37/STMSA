# STMSA

##  Requirements
tensorflow-gpu == 1.14.0  
torch  ==  1.2.0  
transformers == 2.10.0  
numpy == 1.15.4  
sklearn  
python 3.6.13

## Usage

- Use `Dataset.py` to obtain the `train.pkl` and `test.pkl`.
- Use `Dataset_forMissing.py` to obtain the missing modality training data `m11train3.pkl` and missing modality test data `m11test3.pkl`.
- Use `train.py` to train and test the STMSA.
- The dataset can be obtained from [*Baidu Netdisk*](https://pan.baidu.com/s/1lZprgnT3nEntC-iAZOcCiw?pwd=u92i).
